# Sources and verification notes

Public references checked while preparing this package:

- NetEase UU OpenWrt installer examples using `install.sh openwrt $(uname -m)`.
- Community OpenWrt UU installation posts documenting `kmod-tun` requirements.
- ttc0419/uuplugin GitHub project documenting OpenWrt package approaches for IPK/APK package managers.
- OpenWrt documentation for package management and network troubleshooting context.

Important editorial note: the article intentionally avoids promising support for every GL.iNet model. It frames compatibility around OpenWrt, architecture support, storage, and `/dev/net/tun` availability.
